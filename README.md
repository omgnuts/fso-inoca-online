## Key Features
- Antihack
- World Editors
- Admin Features (@ban, @warp, @mute, @unmute, @jail, @filters etc)
- Trace Logging 
- Password reset via email
- Map Restrictions

## Game Play
- Social (Buddy List, World Chat, Private Messaging, Guilds)
- PVE/PVP
- Item Enchantment
- Monster Drops
- NPC Quests
- NPC Shops (buy/sell)
- Player Banking
- Player Housing
- Hundreds of Items
- Container storage
- Factions

## Game Events
- Marriage, Marriage Announcements
- Capture the flag
- Lottery
- Treasure hunt
