BANKING
To use the bank use these commands: /balance, /deposit amount and /withdraw amount. For example, if you wish to deposit 300g you would type /deposit 300 into the chat window. You should always deposit the majority of your money into the bank so as to not risk losing it if you die.

/balance - will display how much money you have in the bank. You must be inside a bank to use this command.
/deposit amount - will allow you to deposit money into a bank. You must be inside a bank to use this command.
/withdraw amount - will allow you to withdraw money from a bank. You must be inside a bank to use this command.

There are vaults located within most banks. You can use these to store excess items. If you use a vault inside a bank, no one but you will be able to retrieve your items.
